## Currently Building

### CampusRent
A production-deployed peer-to-peer campus rental platform focused on real booking workflows.

#### Core Highlights
- 8-stage rental lifecycle engine
- Conflict-safe booking overlap prevention
- Role-guarded state transitions
- JWT authentication + protected APIs
- Prisma ORM + Neon PostgreSQL
- React frontend + Express backend
- Vercel + Render deployment pipeline

> The interesting part for me is solving workflow complexity — making systems predictable, safe, and scalable beyond simple CRUD.
